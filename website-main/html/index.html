Project Overview

In hospitals and pharmaceutical labs, controlled substances like opioids and research materials require secure access to prevent misuse, theft, or tampering.

PharmaGate is a smart double-door security system that introduces multi-layer authentication and environmental awareness for high-security zones. The system combines app authentication, biometric verification (face and fingerprint), tailgating prevention, and fire-response mechanisms. It also includes real-time data monitoring and visualization using ThingSpeak, ensuring security logs are transparent and auditable.

By enhancing physical and digital access control, PharmaGate improves both operational safety and regulatory compliance in sensitive medical environments.

How It Works

PharmaGate is an advanced two-door security system designed for high-security areas, such as hospitals and pharmaceutical labs. The first door features user verification through email and password using a mobile app connected to Firebase. The user then enters a code displayed on the LCD screen above the door. The first door also contains a PIR sensor which turns the LCD off if it does not detect anyone so as to save power. Upon entering the correct code, the door opens via a servomotor. Once the user enters the first door, an IR sensor ensures there is no tailgating. If tailgating is detected, a buzzer will sound, and the second door will remain locked. If no tailgating is detected, the user proceeds to the second door, which requires biometric authentication, including face ID and fingerprint scanning. An ultrasonic sensor detects the distance between the user and the face recognition camera. A LED indicator lights up red if the user is too close or too far from the camera, and turns green when the user is at the correct distance. If both biometric sensors confirm the user's identity, the door opens using a servomotor. To exit, the user scans their RFID card. If authenticated through Firebase, the door opens via the servomotor. In case of a fire, the temperature and humidity sensor triggers an emergency system, and the user must press a button to unlock all doors. Additionally, the data from the temperature and humidity sensor, as well as the entry and exit records of users, are logged in ThingSpeak for secure visualization, traceability, and event monitoring.

Future Scope

The current implementation of PharmaGate establishes a solid foundation for secure access control in critical environments. Below are some areas we envision expanding into for future iterations.

  • Expand biometric options with voice recognition or retina scan.
  • Integrate mobile notifications and alert escalation protocols.
  • Implement access analytics and behavior pattern recognition using AI.
  • Enable dynamic access privileges based on time or location.
  • Support large-scale hospital integration with centralized dashboards.
